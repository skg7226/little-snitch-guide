## Little Snitch 6 的主要功能

**网络连接监控**

实时显示所有出站连接，并在世界地图上标出服务器位置。你家 App 在连美国东海岸还是欧洲某角落，一目了然。端口、协议、连接路径，全都能看到。

**连接警报**

App 想联网时，弹窗提醒你。可以选择"只允许这次"、"永远允许"、"永远拒绝"，也可以直接定义规则批量处理同类请求。如果不想每次都被打扰，还有静默模式，自动记录、事后再审。

**黑名单订阅**

Little Snitch 6 的一大亮点。内置精选黑名单订阅源，直接导入就能在 IP 层面精准拦截广告追踪、恶意域名——不需要你手动一条条添加规则。

**DNS 加密**

支持 DoH（DNS over HTTPS）和 DoT（DNS over TLS），DNS 请求全程加密，防止被运营商或中间人窃听、篡改。

**网络连接分析**

连接记录可以按 App、按域名分组查看，支持高级搜索和筛选，长时间数据留存，方便你回头排查某个 App 的历史行为。

**菜单栏工具**

常驻菜单栏，随时查看实时流量和最近活动，一键切换"警报模式"和"静默模式"，不用每次都打开主界面。
